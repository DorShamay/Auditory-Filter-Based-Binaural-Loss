This repository provides a differentiable PyTorch implementation of an auditory filter–based binaural loss function, based on the auditory model and interaural cue derivation proposed in [1].

## Reference

[1] M. Dietz, S. D. Ewert, and V. Hohmann, “Auditory model-based direction estimation of concurrent speakers from binaural signals,” *Speech Communication*, vol. 53, no. 5, pp. 592–605, 2011.
